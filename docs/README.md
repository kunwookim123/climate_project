## 📝 주요 개념 요약

### 데이터

```
기상 데이터 (2020-2024):
- 174,829건 (97개 지역 × 5년)
- 일강수량, 일사량, 기온, 풍속 등

발전량 데이터:
- PR고정: 발전량 = 일사량 × 20.835
- PR가변: 발전량 = f(일사량, 기온)
```

### 핵심 문제

```
현재 분석:
  강수량 → 발전량 ❌ (잘못된 인과관계)

실제 관계:
  강수량 → 일사량 → 발전량 ✅ (간접 효과)
           └→ 기온 → PR효율
```

### 개선 방안

```
1순위: 장마철 경제적 손실 분석 (실용적)
2순위: PR가변 데이터 활용 (현실적)
3순위: 경로 분석 (학술적)
```
